2:- Project setUp  
1-create workSpace  folder ReactNative  
2- npm install -g expo-cli  
3- expo init MyNativePortfolio  
OR  
npx create-expo-app MyNativePortfolio    
4- cd MyNativePortfolio   
5- npm run android   
OR npx expo start  
OR npm start
